# Руководство по новому интерактивному HUD Guitar Life v13

## Обзор

Новый HUD (Head-Up Display) предоставляет полный интерактивный интерфейс для управления всеми параметрами приложения в реальном времени. Больше не нужно перезапускать приложение для изменения настроек!

## Возможности

### 📊 Интерактивные слайдеры:
- **Тик (5-360 мс)** - скорость обновления автомата
- **RMS сила (0-200%)** - влияние громкости на цвета
- **Макс возраст (10-300)** - максимальный возраст клеток
- **Начало увядания (10-200)** - когда клетки начинают тускнеть
- **Порог очистки (0.001-0.02)** - минимальный RMS для активности
- **RMS мин/макс** - диапазон для цветовой палитры
- **Soft Kill (0-100%)** - процент клеток для мягкого удаления
- **Fade Floor (0.1-1.0)** - минимальная яркость при затухании
- **Pitch Tick Мин/Макс** - диапазон тиков для привязки к высоте

### 🎛️ Кнопки включения/выключения:
- **Pitch→Тик** - привязка скорости тика к высоте звука
- **RMS Мод** - модуляция цветов по громкости
- **Soft Clear** - мягкая очистка поля
- **Kill/Fade** - режим Soft Clear (удаление vs затухание)
- **Mirror X/Y** - зеркальное отражение по осям

### ✨ FX эффекты:
- **Trails** - следы/шлейфы
- **Blur** - размытие
- **Bloom** - свечение
- **Poster** - постеризация
- **Dither** - дизеринг
- **Scans** - сканлайны
- **Pixel** - пикселизация
- **Line** - контуры

### 🎮 Быстрые действия:
- **Random** - генерация случайных стабильных паттернов
- **Clear** - очистка всех слоев
- **Test** - создание тестового блока 2x2
- **Joy Division** - предустановка в стиле Joy Division

## Управление

### Клавиатура:
- `H` - показать/скрыть HUD
- `Tab` (удерживать) - мини-режим HUD
- `ESC` - выход из приложения
- `R` - случайные паттерны
- `C` - очистка
- `T` - тест
- `F1` - переключить Trails
- `F2` - переключить Blur
- `F3` - применить Joy Division

### Мышь:
- **Кликните по кнопке "Развернуть"** в нижней части HUD для доступа к полной панели управления
- **Перетаскивайте слайдеры** для изменения числовых параметров
- **Кликайте кнопки** для переключения опций и выполнения действий

## Расположение

HUD располагается в правой части экрана с полупрозрачным фоном. В свернутом виде показывает только базовую информацию. В развернутом виде предоставляет доступ ко всем 40+ параметрам.

## Обратная связь

Все изменения применяются мгновенно. Консоль выводит отладочную информацию о изменениях параметров для мониторинга.
