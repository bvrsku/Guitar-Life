# 🔧 Исправление отображения множественных слоев

## 🎯 Проблема:
При наличии нескольких слоев на экране отображается только один слой, вместо наложения всех слоев друг на друга.

## 🔍 Причина:
Функция `build_color_image()` создавала RGB изображения (3 канала) без альфа-канала, из-за чего пустые области одного слоя полностью перезаписывали другие слои вместо того чтобы быть прозрачными.

## ✅ Исправления:

### 1. **Добавлен альфа-канал в `build_color_image()`**
```python
# Было: RGB (3 канала)
img = np.zeros((H, W, 3), dtype=np.uint8)

# Стало: RGBA (4 канала)  
img = np.zeros((H, W, 4), dtype=np.uint8)
img[i, j, :3] = color  # RGB цвет
img[i, j, 3] = 255     # Альфа = непрозрачность для живых клеток
# Пустые области остаются с alpha=0 (прозрачные)
```

### 2. **Обновлена функция `blit_layer()` для RGBA**
```python
# Обработка RGBA изображений с альфа-каналом
if color_img.shape[2] == 4:
    surf = pygame.surfarray.make_surface(np.transpose(color_img[:,:,:3], (1, 0, 2)))
    surf = surf.convert_alpha()
    surf_alpha = pygame.surfarray.pixels_alpha(surf)
    surf_alpha[:] = np.transpose(color_img[:,:,3])
    
# Правильное альфа-смешивание
surf.set_blendmode(pygame.BLEND_ALPHA_SDL2)
```

### 3. **Добавлена отладочная информация**
```python
print(f"DEBUG render: Total layers={len(self.layers)}, rendering={len(layers)}")
for i, L in enumerate(layers):
    live_cells = np.sum(L.grid)
    print(f"DEBUG render: Layer {i} ({L.rule}): {live_cells} live cells")
```

- ✅ **Пустые области прозрачны** - не перезаписывают другие слои
- ✅ **Живые клетки непрозрачны** - правильно накладываются
- ✅ **Solo/Mute режимы работают** для изоляции отдельных слоев
