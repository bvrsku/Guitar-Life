# 🚀 Как запустить Guitar Life - Структурированная версия

## 📋 Доступные версии

### 1. `main.py` - Базовая структурированная версия
**Простая демонстрация основных возможностей**

**Возможности:**
- ✅ Клеточные автоматы (Conway, HighLife, Day&Night)
- ✅ Яркая цветовая схема на основе возраста клеток
- ✅ Интерактивное рисование мышью
- ✅ Отображение статистики (поколение, клетки, FPS)
- ✅ Начальный спавн клеток
- ✅ Сетка для лучшей видимости

**Управление:**
- `SPACE` - пауза/продолжение
- `C` - очистка поля
- `R` - случайный спавн
- `1,2,3` - смена правил
- `Мышь` - рисование клеток
- `ESC` - выход

### 2. `main_enhanced.py` - Расширенная версия
**Демонстрация всех возможностей структурированных модулей**

**Дополнительные возможности:**
- ✅ 43 цветовые палитры (переключение на `P`)
- ✅ 8 методов спавна (переключение на `TAB`)
- ✅ 9 правил автоматов (клавиши `1-4`)
- ✅ Переключаемая сетка (`G`)
- ✅ Поддержка эффектов (`E`)
- ✅ Рисование правой кнопкой мыши (удаление)
- ✅ Расширенная статистика

**Управление:**
- `SPACE` - пауза/продолжение
- `C` - очистка поля
- `R` - случайный спавн
- `G` - включить/выключить сетку
- `E` - включить/выключить эффекты
- `1,2,3,4` - смена правил автоматов
- `TAB` - смена метода спавна
- `P` - смена цветовой палитры
- `ЛКМ` - добавить клетки
- `ПКМ` - удалить клетки
- `ESC` - выход

### 3. `demo_structured.py` - Демонстрация модулей
**Показывает возможности каждого модуля без GUI**

### 4. `GuitarLife_v21.py` - Оригинальная полная версия
**Исходная монолитная версия со всеми функциями**

## 🛠️ Установка и запуск

### Требования
```bash
pip install pygame numpy
```

### Опционально (для аудио):
```bash
pip install sounddevice librosa
```

### Запуск
```bash
# Базовая версия (рекомендуется для начала)
python main.py

# Расширенная версия (все возможности)
python main_enhanced.py

# Демонстрация модулей
python demo_structured.py

# Оригинальная версия
python GuitarLife_v21.py
```

## 🎨 Цветовые палитры (main_enhanced.py)

Доступные палитры (переключение на `P`):
- **Fire** - огненная палитра (красный→оранжевый→желтый)
- **Ocean** - океанская (синий→голубой→белый)
- **Galaxy** - галактическая (черный→фиолетовый→синий→белый)
- **Aurora** - северное сияние (зеленый→синий→фиолетовый)
- **Sunset** - закатная (фиолетовый→розовый→оранжевый)
- **Spring** - весенняя (зеленый→желтый→розовый)
- И многие другие...

## 🤖 Правила клеточных автоматов

### Conway (классическая Жизнь)
- **Рождение**: 3 соседа
- **Выживание**: 2-3 соседа
- **Особенности**: стабильные структуры, глайдеры

### HighLife
- **Рождение**: 3 или 6 соседей
- **Выживание**: 2-3 соседа
- **Особенности**: репликаторы, более динамичные паттерны

### Day&Night
- **Рождение**: 3,6,7,8 соседей
- **Выживание**: 3,4,5,6,7,8 соседей
- **Особенности**: симметричное правило, инверсия паттернов

### Seeds
- **Рождение**: 2 соседа
- **Выживание**: никогда
- **Особенности**: клетки живут только один ход

## 🎲 Методы спавна (main_enhanced.py)

Переключение на `TAB`:
1. **Случайные точки** - классический случайный спавн
2. **Стабильные блоки** - блоки 2x2 (стабильные структуры)
3. **Глайдеры** - движущиеся паттерны
4. **Осцилляторы** - мигающие структуры
5. **Смешанный** - комбинация разных типов
6. **Линии** - горизонтальные и вертикальные
7. **Кресты** - крестообразные паттерны
8. **Кольца** - круглые структуры

## 📊 HUD информация

### Статистика
- **Generation** - номер поколения
- **Rule** - текущее правило автомата
- **Cells** - количество живых клеток
- **FPS** - кадры в секунду
- **Palette** - текущая цветовая палитра (enhanced)
- **Status** - состояние (RUNNING/PAUSED)

### Индикаторы состояния
- 🟢 **RUNNING** - симуляция активна
- 🟡 **PAUSED** - симуляция на паузе
- 🔵 **Статистика** - голубой цвет
- 🟣 **Палитра** - фиолетовый цвет
- 🟢 **Правило** - зеленый цвет

## 🐛 Решение проблем

### Черный экран
- Убедитесь, что есть живые клетки (нажмите `R` для спавна)
- Проверьте, что симуляция не на паузе (нажмите `SPACE`)
- Попробуйте нарисовать клетки мышью

### Низкая производительность
- Уменьшите количество клеток (нажмите `C` для очистки)
- Используйте более простые правила (Conway вместо Day&Night)
- Закройте другие приложения

### Ошибки импорта
```bash
# Установите зависимости
pip install pygame numpy

# Для аудио функций (опционально)
pip install sounddevice librosa
```

### Проблемы с аудио
- Аудио функции опциональны
- Приложение работает без sounddevice и librosa
- При ошибках аудио модуль отключается автоматически
