Расширенный HUD Guitar Life v13 - Список изменений

✅ ОСНОВНЫЕ ВОЗМОЖНОСТИ:
1. Прокрутка содержимого HUD (колесо мыши, клавиши Up/Down)
2. Выбор палитры из выпадающего списка
3. Выбор правил автомата из выпадающего списка
4. Визуальные разделители для группировки параметров

✅ ПАРАМЕТРЫ В HUD:
Основные параметры:
- Tick (миллисекунды): 5-360 мс
- RMS Power: 0-200
- RMS Background: 0-100
- RMS Frequency: 0-100

Пороги RMS:
- RMS Threshold: 0-100
- RMS Soft Start: 0-100
- RMS Soft High: 0-100

Параметры Pitch:
- Pitch Tick Range: 2-100

Параметры эффектов:
- Trail Strength: 0.01-0.2 (сила следа)
- Blur Scale: 1-10 (размытие)
- Bloom Strength: 0.1-1.0 (свечение)
- Posterize Levels: 2-16 (постеризация)
- Scanline Strength: 0.0-1.0 (развертка)
- Pixel Block Size: 1-8 (пикселизация)
- Outline Thickness: 0.0-5.0 (контур)

Управление:
- Кнопки Play/Pause/Stop
- Сброс автоматов
- Очистка следов

Выбор:
- Палитра (выпадающий список)
- Правила автомата (выпадающий список)

✅ VISUAL IMPROVEMENTS:
- Организованная структура с разделителями
- Полная прокрутка всех элементов
- Синхронизация параметров с основным приложением
- Правильное масштабирование и позиционирование

✅ ТЕХНИЧЕСКИЕ ОСОБЕННОСТИ:

- Поддержка прокрутки с сохранением координат
- Автоматическое обновление значений из приложения

✅ КОМАНДЫ УПРАВЛЕНИЯ:
