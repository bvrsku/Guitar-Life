## Новые возможности

### Управление слоями:
- **Генерация слоев**: Можно создавать от 1 до 5 слоев с различными параметрами
- **Пресеты**: Предустановленные конфигурации (balanced, chaotic, stable)
- **Дублирование**: Копирование существующих слоев с небольшими изменениями
- **Циклическое переключение правил**: Смена алгоритмов клеточных автоматов

### Паттерны:
- **Тестовые паттерны**: Для проверки работы системы
- **Случайные паттерны**: Генерация разнообразных стартовых конфигураций
- **Планеры Conway**: Классические движущиеся паттерны
- **Различные типы очистки**: Полная, частичная, по возрасту

### Сохранение/загрузка:
- **Пресеты**: Сохранение текущих настроек в JSON файлы
- **Восстановление**: Загрузка ранее сохраненных конфигураций
- **Конфигурация по умолчанию**: Автоматическое создание базовых настроек
