# Добавленные функции в Guitar Life HUD

## Новые параметры в HUD

### 1. Aging Speed (Скорость старения)
- **Отображение**: `Aging Speed: X.Xx` (где X.X - множитель скорости)
- **Описание**: Показывает текущую скорость старения клеток
- **Значения**: 0.1x - 5.0x (по умолчанию 1.0x)
- **Эффект**: 
  - 1.0x = нормальная скорость старения (клетки стареют на 1 каждый тик)
  - 2.0x = двойная скорость (клетки стареют на 2 каждый тик)
  - 0.5x = половинная скорость (клетки стареют вероятностно)

### 2. Alpha Values (Значения прозрачности слоев)
- **Отображение**: `Alpha Values: L0: 255/255 | L1: 200/180 | L2: 150/100`
- **Формат**: `LN: live/old` где:
  - `LN` - номер слоя (L0, L1, L2, ...)
  - `live` - прозрачность живых клеток (0-255)
  - `old` - прозрачность старых клеток (0-255)
- **Разделитель**: ` | ` между слоями

## Где в коде

### Aging Speed
- **Инициализация**: `self.aging_speed = sel.get('aging_speed', 1.0)` (строка ~2475)
- **Логика старения**: В методе `update_layers()` (строка ~3085)
- **Обработка изменений**: В методе `on_hud_parameter_change()` (строка ~2553)
- **Отображение**: В основном игровом цикле (строка ~3292)

### Alpha Values
- **Формирование**: В основном игровом цикле (строки ~3281-3284)
- **Отображение**: В info словаре (строка ~3293)

## Технические детали

- Aging Speed поддерживает дробные значения с вероятностным старением
- Alpha Values показывают реальные значения прозрачности каждого слоя
- Оба параметра обновляются в реальном времени при изменении через HUD
