## Как использовать

### Отключить RMS палитру полностью:
1. Установите `RMS Mode = Disabled`
   ИЛИ
2. Нажмите кнопку `RMS Enabled` чтобы выключить

### Классическое поведение (как раньше):
- `RMS Mode = Brightness`
- `RMS Enabled = включено`

### Смешивание двух палитр:
- `RMS Mode = Palette`
- `Blend Mode = выберите подходящий (Normal, Additive, Screen, etc.)`
- `RMS Enabled = включено`

## Режимы блендинга

- **Normal**: Альфа-блендинг (плавное смешивание)
- **Additive**: Аддитивное смешивание (ярче)
- **Screen**: Осветление
- **Multiply**: Затемнение
- **Overlay**: Контрастное смешивание
