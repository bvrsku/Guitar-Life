## 🎛️ Реализованные параметры

### 1. Clear Type (Тип очистки)
**Расположение**: HUD → Раздел "ACTIONS" → Комбобокс "Clear Type"

**Доступные варианты**:
- `Полная очистка` - удаляет все клетки со всех слоев
- `Частичная очистка` - удаляет 50% клеток случайным образом  
- `Очистка по возрасту` - удаляет клетки старше определенного возраста
- `Случайная очистка` - удаляет случайный процент клеток (20-40%)

**Программный доступ**:
```python
app.on_hud_parameter_change('clear_type', 'Полная очистка')
app.clear_with_type()  # Выполнить очистку выбранного типа
```

### 2. Soft Mode (Мягкий режим)
**Расположение**: HUD → Раздел "CONTROLS" → Комбобокс "Soft Mode"

**Доступные варианты**:
- `Kill` → Удалять клетки (мгновенная смерть)
- `Fade` → Затухание клеток (постепенное исчезновение) 
- `Fade+Kill` → Затухание + удаление (комбинированный режим)

**Программный доступ**:
```python
app.on_hud_parameter_change('soft_mode', 'Fade')
# Внутреннее значение: app.soft_mode = 'Затухание клеток'
```

## 🔧 Техническая реализация

### Интеграция в HUD
```python
# В классе HUD
self.comboboxes['clear_type'] = UIComboBox(
    x=self.x_actions, y=combo_y, w=combo_width, h=combo_height,
    options=['Полная очистка', 'Частичная очистка', 'Очистка по возрасту', 'Случайная очистка'],
    on_change=lambda v: self.on_change('clear_type', v)
)

self.comboboxes['soft_mode'] = UIComboBox(
    x=self.x_controls, y=combo_y, w=combo_width, h=combo_height,
    options=['Kill', 'Fade', 'Fade+Kill'],
    on_change=lambda v: self.on_change('soft_mode', v)
)
```

### Обработка изменений
```python
def on_hud_parameter_change(self, param_name: str, value):
    if param_name == 'soft_mode':
        # Преобразуем значение комбобокса в внутреннее представление
        if value == "Kill":
            self.soft_mode = 'Удалять клетки'
        elif value == "Fade":
            self.soft_mode = 'Затухание клеток'
        elif value == "Fade+Kill":
            self.soft_mode = 'Затухание + удаление'
    elif param_name == 'clear_type':
        self.clear_type = str(value)
```
