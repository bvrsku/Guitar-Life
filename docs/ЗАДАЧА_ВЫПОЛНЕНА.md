## 🚀 Новые возможности для пользователя

### Управление количеством слоев
- **➕ Add Layer** - динамическое добавление слоев (максимум 5)
- **➖ Remove** - безопасное удаление с сохранением минимум 1 слоя
- **🧹 Clear** - быстрая очистка выбранного слоя

### Массовые операции
- **🎭 Solo All / Unsolo All** - управление solo режимом всех слоев
- **🔇 Mute All / Unmute All** - включение/отключение всех слоев

### Визуализация
- **📊 Статус слоев** - детальная информация по каждому слою
- **🎯 Индикаторы** - визуальные маркеры состояния solo/mute
- **📈 Статистика** - количество клеток, альфа значения
