# Новые параметры очистки в HUD

## Добавленные параметры

Были добавлены три новых настраиваемых параметра очистки в интерфейс Guitar Life:

### 1. **Partial Clear (%)** - Процент частичной очистки
- **Диапазон**: 10-90%
- **По умолчанию**: 50%
- **Описание**: Определяет какой процент клеток будет удален при выборе типа очистки "Частичная очистка"
- **Использование**: Позволяет точно контролировать интенсивность частичной очистки

### 2. **Age Threshold** - Порог возраста для очистки
- **Диапазон**: 1-50 единиц
- **По умолчанию**: 10
- **Описание**: Определяет минимальный возраст клеток для удаления при выборе типа очистки "Очистка по возрасту"
- **Использование**: Позволяет сохранять молодые клетки и удалять только старые

### 3. **Random Clear (%)** - Процент случайной очистки
- **Диапазон**: 10-90%
- **По умолчанию**: 30%
- **Описание**: Определяет какой процент клеток будет случайно удален при выборе типа очистки "Случайная очистка"
- **Использование**: Контролирует интенсивность случайного удаления клеток

## Расположение в GUI

Все три слайдера находятся в правой панели HUD в секции параметров популяции, после существующих слайдеров:
- Max Cells (%)
- Clear At (%)
- Age Bias (%)
- **→ Partial Clear (%)** ← НОВЫЙ
- **→ Age Threshold** ← НОВЫЙ  
- **→ Random Clear (%)** ← НОВЫЙ

## Как использовать

1. **Выберите тип очистки** в комбобоксе "Clear Type":
   - Полная очистка
   - Частичная очистка
   - Очистка по возрасту
   - Случайная очистка

2. **Настройте соответствующий параметр**:
   - Для "Частичная очистка" → используйте "Partial Clear (%)"
   - Для "Очистка по возрасту" → используйте "Age Threshold"
   - Для "Случайная очистка" → используйте "Random Clear (%)"

3. **Нажмите "Clear"** для выполнения очистки с выбранными параметрами

## Техническая реализация

- ✅ Параметры сохраняются в конфигурации приложения
- ✅ Значения синхронизируются между GUI и логикой приложения
- ✅ Параметры сбрасываются к значениям по умолчанию при нажатии "Reset"
- ✅ Изменения применяются в реальном времени
- ✅ Все типы очистки используют соответствующие настраиваемые параметры

## Пример использования

```python
# Настройка в коде
app.clear_partial_percent = 75    # 75% клеток при частичной очистке
app.clear_age_threshold = 15      # Удалять клетки старше 15 единиц
app.clear_random_percent = 40     # 40% случайных клеток

# Выполнение очистки
app.clear_type = "Частичная очистка"
app.clear_with_type()  # Удалит 75% клеток
```
