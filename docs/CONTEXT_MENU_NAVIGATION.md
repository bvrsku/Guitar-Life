# Руководство по навигации в контекстных меню

## ✨ Новые возможности навигации

В контекстных меню (выпадающих списках) для выбора палитр, правил автомата и других параметров теперь доступна полная навигация с клавиатуры.

## 🎮 Клавиши управления

### Основная навигация
- **↑/↓** - Перемещение по элементам списка
- **Enter** или **Space** - Выбор текущего выделенного элемента
- **Escape** - Закрытие меню без изменений

### Быстрая навигация
- **Home** - Переход к первому элементу списка
- **End** - Переход к последнему элементу списка
- **Page Up** - Прокрутка на страницу вверх
- **Page Down** - Прокрутка на страницу вниз

### Быстрый поиск
- **Ввод текста** - Поиск элементов по начальным символам
- **Backspace** - Удаление последнего символа поиска
- Поиск автоматически очищается через 2 секунды

### Мышь
- **Колесо мыши** - Прокрутка в открытом меню
- **Клик мыши** - Выбор элемента или закрытие меню

## 🎨 Визуальные индикаторы

- **Синий фон** - Выбранный в данный момент элемент
- **Серый фон с синей рамкой** - Выделенный клавиатурой элемент
- **Светло-серый фон** - Элемент под курсором мыши
- **Окошко поиска** - Показывается внизу при вводе текста

## 🚀 Примеры использования

1. **Быстрый выбор палитры**:
   - Откройте выпадающий список палитр
   - Начните вводить "rain" для поиска "Rainbow"
   - Нажмите Enter для выбора

2. **Навигация по правилам автомата**:
   - Откройте список правил
   - Используйте ↑/↓ для просмотра
   - Нажмите Space для выбора

3. **Быстрый переход**:
   - Нажмите Home для перехода к началу списка
   - Нажмите End для перехода к концу

## 🔧 Техническая информация

Функциональность реализована в классе `UIComboBox` в файле `guitar_life_patched_HUD (1).py`:
- Обработка событий клавиатуры в методе `handle_event()`
- Визуальная отрисовка выделения в методе `draw()`
- Автоматическая прокрутка для видимости выделенного элемента
- Система поиска с таймером автоочистки

## 📝 Примечания

- Поиск работает по начальным символам элементов (префиксный поиск)
- Поиск нечувствителен к регистру
- Автоматическая прокрутка обеспечивает видимость выделенного элемента
- Все горячие клавиши работают только при открытом контекстном меню
