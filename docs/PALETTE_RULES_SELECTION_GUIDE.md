# 🎨 Руководство по выбору палитр и правил в HUD

## 🎛️ ПОЛНЫЙ ФУНКЦИОНАЛ HUD

Guitar Life теперь предоставляет **полный интерактивный контроль** над всеми параметрами через HUD с возможностью выбора палитр и правил автоматов.

### 📋 КАТЕГОРИИ HUD:

#### 🔊 **AUDIO** (7 параметров)
- **gain**: Усиление звука (0.1-10.0)
- **rms_strength**: Сила RMS влияния (0-500%)
- **color_rms_min/max**: Диапазон RMS для цвета
- **pitch_tick_enable**: Высота звука влияет на скорость
- **pitch_tick_min/max**: Диапазон времени тика

#### 🎨 **VISUAL** (8 параметров)
- **global_v_mul**: Глобальная яркость (0.1-3.0)
- **hue_offset**: Сдвиг оттенка (0-360°)
- **aging_speed**: Скорость старения (0.1-10.0)
- **fade_start**: Начало затухания (1-500)
- **fade_sat/val_drop**: Падение насыщенности/яркости (0-100%)
- **max_age**: Максимальный возраст (1-1000)
- **palette_invert**: Инверсия палитр

#### 🏗️ **LAYERS** (11 параметров)
**Основные контролы:**
- **layer_count**: Количество слоев (1-5)
- **L1, L2, L3, L4, L5**: Кнопки выбора активного слоя

**Контролы выбранного слоя:**
- **Rule**: Правило автомата
- **Age Palette**: Палитра для возраста
- **RMS Palette**: Палитра для RMS
- **Alpha**: Прозрачность слоя (0.0-1.0)
- **Blend Mode**: Режим смешивания
- **SOLO/MUTE**: Изоляция/отключение слоя
- **Mirror X/Y**: Зеркалирование

#### ✨ **EFFECTS** (8 параметров)
- Scanlines, Posterize, Dither, Pixelate
- Blur с настройками интенсивности

#### ⚡ **PERF** (6 параметров)
- Скорость обновления и контроль популяции

#### 🎬 **ACTIONS** (14 параметров)
- Очистка, сохранение, загрузка пресетов
- Спецэффекты и паттерны

## 🎮 ПРАВИЛА АВТОМАТОВ

### 📝 **Доступные правила (CA_RULES):**

| Правило | Описание | Характеристики |
|---------|----------|----------------|
| **Conway** | Классическая "Игра Жизни" | Стабильные структуры, планеры |
| **HighLife** | Модификация Conway с дополнительным рождением | Больше структур |
| **Day&Night** | Обращенные правила | Инверсное поведение |
| **Replicator** | Самовоспроизводящиеся структуры | Быстрое размножение |
| **Seeds** | Только рождение, нет выживания | Взрывной рост |
| **Maze** | Создает лабиринтоподобные структуры | Сложные паттерны |
| **Coral** | Рост подобный кораллам | Органические формы |
| **LifeWithoutDeath** | Клетки никогда не умирают | Постоянный рост |
| **Gnarl** | Хаотичное поведение | Турбулентность |

### 🎯 **Как выбрать правило:**
1. **Перейдите в категорию LAYERS** (клавиша 3 или клик)
2. **Выберите слой** кнопками L1-L5
3. **Откройте выпадающий список "Rule"**
4. **Выберите желаемое правило** из списка

## 🎛️ ИНТЕРАКТИВНОЕ УПРАВЛЕНИЕ

### 🖱️ **Навигация по HUD:**

1. **Переключение категорий:**
   - **Клавиши 1-6** или **клики по вкладкам**
   - AUDIO(1) → VISUAL(2) → LAYERS(3) → EFFECTS(4) → PERF(5) → ACTIONS(6)

2. **Выбор слоя:**
   - В категории **LAYERS** используйте кнопки **L1, L2, L3, L4, L5**
   - Активная кнопка слоя подсвечивается
   - Все остальные контролы применяются к выбранному слою

3. **Работа с палитрами:**
   - **Age Palette** - влияет на цвет в зависимости от возраста клетки
   - **RMS Palette** - влияет на цвет в зависимости от уровня звука
   - Выпадающие списки с полным набором палитр

4. **Режимы смешивания:**
   - **normal** - обычное наложение
   - **additive** - аддитивное смешивание
   - **screen** - осветляющее смешивание
   - **multiply** - умножающее смешивание
   - **overlay** - наложение
