# Добавленные настройки удаления клеток в HUD

## 📊 Резюме изменений

Успешно добавлены все настройки системы удаления клеток в HUD для удобного управления популяцией.

### 🎛️ Добавленные слайдеры в секции "SOFT CLEAN":

1. **Soft Kill (%)** - `soft_kill_rate` (0-100, по умолчанию 80)
   - Процент клеток для удаления при срабатывании системы

2. **Fade Floor** - `soft_fade_floor` (0.1-1.0, по умолчанию 0.3) 
   - Минимальная яркость при режиме затухания

3. **Fade Down** - `soft_fade_down` (1-10, по умолчанию 1) ⭐ НОВЫЙ
   - Скорость затухания вниз (как быстро яркость уменьшается)

4. **Fade Up** - `soft_fade_up` (1-20, по умолчанию 5) ⭐ НОВЫЙ  
   - Скорость затухания вверх (как быстро яркость восстанавливается)

5. **Max Cells (%)** - `max_cells_percent` (10-90, по умолчанию 50)
   - Максимальный процент заполнения сетки клетками

6. **Clear At (%)** - `soft_clear_threshold` (40-95, по умолчанию 70)
   - Порог заполнения, при котором начинается мягкое удаление

7. **Age Bias (%)** - `age_bias` (0-100, по умолчанию 80)
   - Предпочтение возраста при выборе клеток для удаления

### 🔘 Кнопки управления:

1. **Soft Clear** - `soft_clear_enable` (по умолчанию ВКЛ)
   - Включение/выключение всей системы удаления

2. **Kill/Fade** - `soft_mode_toggle` (по умолчанию Kill) ⭐ НОВЫЙ
   - Переключение между режимами:
   - Kill (OFF): Удалять клетки полностью
   - Fade (ON): Затухание клеток через изменение яркости

### 📋 Информационные панели:

1. **Population Info** - отображает текущую популяцию
   - Формат: "Population: 1250 / 8400 (14.9%)"

2. **Removal Info** - отображает статус системы удаления ⭐ НОВАЯ
   - Формат: "Removal: ON | Kill | Threshold: 70% | Max: 50%"

## 🔧 Техническая реализация:

### Добавленные слайдеры:
- `self.sliders['soft_fade_down']` - управление скоростью затухания вниз
- `self.sliders['soft_fade_up']` - управление скоростью затухания вверх

### Добавленная информационная панель:
- `self.labels['removal_info']` - отображает текущие настройки системы удаления

### Добавленная обработка параметров:
- `soft_fade_down` в `set_parameter()` и `update_from_app()`
- `soft_fade_up` в `set_parameter()` и `update_from_app()`

### Логика работы системы удаления:

1. **Проверка порогов:**
   - Если клеток < `soft_clear_threshold` → удаление не происходит
   - Если клеток >= `soft_clear_threshold` → мягкое удаление
   - Если клеток > `max_cells_percent` → агрессивное удаление

2. **Режимы удаления:**
   - **Kill режим**: Полное удаление клеток по алгоритму
   - **Fade режим**: Уменьшение `global_v_mul` для затухания всех клеток

3. **Алгоритм удаления:**
   - Базовое удаление: `soft_kill_rate` процентов от общего количества
   - Адаптивное удаление при превышении порогов
   - Приоритет старых клеток при `age_bias` > 50

## ✅ Результат:

Теперь пользователь может полностью контролировать систему удаления клеток прямо из HUD:
- Настраивать пороги срабатывания
- Выбирать режим удаления (Kill/Fade) 
- Регулировать интенсивность удаления
- Управлять скоростями затухания
- Видеть текущий статус системы в реальном времени

Все настройки интегрированы в существующий HUD без нарушения пользовательского интерфейса и сохраняют свои значения в конфигурации приложения.
