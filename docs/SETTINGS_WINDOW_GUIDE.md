# Guitar Life - Окно настроек

## Решение проблемы производительности HUD

### Проблема
Оригинальный HUD из модуля `guitar_life` крайне неоптимизирован и тратил **95% времени кадра** (169ms из 178ms), что приводило к FPS всего 5-6 кадров в секунду.

### Решение
Создано **отдельное окно настроек** на базе tkinter, которое работает в отдельном потоке и не влияет на производительность основного приложения.

## Как использовать

### Запуск окна настроек
1. **Запустите Guitar Life**: `python guitar_life_patched.py`
2. **Откройте окно настроек**: нажмите `Ctrl+S` в основном окне
3. **Начните настройку**и: зменения применяются мгновенно!

### Горячие клавиши
- `Ctrl+S` - Открыть окно настроек
- `H` - Показать/скрыть встроенный HUD (для сравнения производительности)
- `ESC` - Выход из приложения

### Возможности окна настроек

#### Основные параметры
- **Tick Interval** - интервал обновления симуляции (мс)
- **RMS Strength** - сила реакции на звук
- **Max Age** - максимальный возраст клеток
- **Aging Speed** - скорость старения

#### Эффекты (FX)
Переключатели для визуальных эффектов:
- Trails (следы)
- Blur (размытие)
- Bloom (свечение)
- Posterize (постеризация)
- Dither (дизеринг)
- Scanlines (строки развертки)
- Pixelate (пикселизация)
- Outline (контуры)

#### Управление слоями
- **Количество слоев** - от 1 до 5 слоев
- Каждый слой может иметь свои настройки

#### Быстрые действия
- **Random Pattern** - создать случайный паттерн
- **Clear** - очистить все слои
- **Test Pattern** - создать тестовый паттерн

#### Статус в реальном времени
- Количество активных слоев
- Общее количество живых клеток
- Счетчик кадров

## Технические детали

### Архитектура
```
┌─────────────────┐    ┌─────────────────┐

│   приложение    │◄──►│   настроек      │

        │                       │
        │              ┌────────▼────────┐
        │              │ Queue для       │
        │              │ передачи        │
        │              │ изменений       │
        └──────────────┤                 │
                       └─────────────────┘
```

### Производительность
- **Основное окно**: 50-60 FPS (16-20ms на кадр)
- **HUD**: 0.1ms (вместо 169ms!)
- **Окно настроек**: работает в отдельном потоке, не влияет на FPS

### Файлы
- `guitar_life_patched.py` - основное приложение
- `settings_window.py` - окно настроек
- Изменения применяются через существующий механизм `on_hud_parameter_change()`

## Переключение режимов HUD

В коде есть переменная `use_simple_hud`:

```python
# Можно переключать:
use_simple_hud = True   # Простой быстрый текстовый HUD (рекомендуется)
use_simple_hud = False  # Полнофункциональный медленный GUI HUD
```

### Простой HUD (use_simple_hud = True)
- ✅ **Очень быстро**: 0.1ms на кадр
- ✅ **Не влияет на FPS**: 50-60 FPS
- ❌ **Только информация**: нет интерактивных элементов
- ✅ **Используйте с окном настроек** для полной функциональности

### Полный HUD (use_simple_hud = False)  
- ❌ **Очень медленно**: 169ms на кадр
- ❌ **Убивает FPS**: 5-6 FPS
- ✅ **Полная функциональность**: все элементы управления
- ❌ **Не рекомендуется** из-за проблем производительности

## Рекомендации

1. **Используйте простой HUD + окно настроек** для лучшей производительности
2. **Откройте окно настроек** при запуске для удобства
3. **Экспериментируйте** с настройками в реальном времени
4. **Следите за статусом** в окне настроек для мониторинга производительности

Теперь Guitar Life работает быстро И удобно! 🎸🎵
