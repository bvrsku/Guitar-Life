# Обновление системы палитр в Guitar Life

## Изменения в разделении палитр

### Старая система
- **"Возраст + RMS"** - комбинированный режим цвета
- **"Только RMS"** - цвет только от уровня RMS
- **"Высота ноты (Pitch)"** - цвет от частоты звука

### Новая система
- **"HSV-дизайны"** - комбинированный режим (бывший "Возраст + RMS")
- **"HSV Палитры"** - только RMS режим (бывший "Только RMS")
- **"Высота ноты (Pitch)"** - остается без изменений

## Технические изменения

### 1. Layer класс
```python
# Обновлен комментарий
color_mode: str  # "HSV-дизайны" | "HSV Палитры" | "Высота ноты (Pitch)"
```

### 2. Функция build_color_image()
```python
# Обновлена логика выбора режима
if mode == "HSV Палитры":
    color = color_from_rms(rms, rms_palette, cmin, cmax, v_mul)
elif mode == "Высота ноты (Pitch)":
    color = color_from_pitch(pitch, rms, rms_strength, v_mul)
else:
    # HSV-дизайны (бывший "Возраст + RMS")
    color = color_from_age_rms(...)
```

### 3. Конфигурации по умолчанию
Все места где использовался `'Возраст + RMS'` заменены на `'HSV-дизайны'`:
- `choose_settings()` - возвращаемые конфигурации
- `modern_gui.py` - дефолтные настройки слоев
- `App.__init__()` - создание новых слоев

## Функциональность режимов

### HSV-дизайны (основной режим)
- ✅ Использует возраст клеток как основу
- ✅ Модифицируется уровнем RMS
- ✅ Поддерживает age_palette и rms_palette
- ✅ Включает эффекты затухания
- ✅ Поддерживает разные режимы смешивания (brightness/palette)

### HSV Палитры (упрощенный режим)
- ✅ Игнорирует возраст клеток
- ✅ Цвет определяется только RMS
- ✅ Использует только rms_palette
- ✅ Подходит для простых визуализаций

### Высота ноты (специальный режим)
- ✅ Цвет зависит от частоты звука
- ✅ Включает модификацию RMS
- ✅ Использует специальную цветовую функцию

## Обратная совместимость

⚠️ **Внимание:** Старые конфигурационные файлы могут содержать устаревшие названия режимов.

### Рекомендуемое преобразование:
```python
def update_legacy_config(config):
    if config.get('color_mode') == 'Возраст + RMS':
        config['color_mode'] = 'HSV-дизайны'
    elif config.get('color_mode') == 'Только RMS':
        config['color_mode'] = 'HSV Палитры'
    return config
```

## Преимущества нового именования

1. **Ясность**: "HSV-дизайны" и "HSV Палитры" лучше описывают функциональность
2. **Единообразие**: Оба режима теперь относятся к HSV цветовому пространству
3. **Расширяемость**: Легче добавлять новые HSV-based режимы в будущем
4. **Интуитивность**: Пользователю понятнее различие между режимами
