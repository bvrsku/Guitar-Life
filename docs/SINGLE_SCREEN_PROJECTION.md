# ПРОЕКЦИЯ МНОЖЕСТВЕННЫХ СЛОЕВ НА ОДИН ЭКРАН

## 🎯 Принцип работы

В Guitar Life **ВСЕ слои проектируются на ОДИН экран** через систему композитного рендеринга.

## 🖥️ Как это работает

### 1. Единый Canvas (Холст)
- Все слои рендерятся в один общий canvas
- Каждый кадр начинается с очистки фона
- Слои накладываются последовательно один поверх другого

### 2. Математика наложения
```
Финальный_пиксель = Фон + (Слой1 × α1) + (Слой2 × α2) + ... + (СлойN × αN)
```

### 3. Контроль прозрачности
- **alpha_live**: Прозрачность живых клеток (0-255)
- **alpha_old**: Прозрачность старых клеток (0-255)
- Позволяет создавать глубину и атмосферу

### 4. Режимы смешивания (blend_mode)
- **normal**: Обычное наложение
- **screen**: Осветление
- **overlay**: Контрастное смешивание  
- **multiply**: Затемнение
- **additive**: Яркое сложение цветов

## 🎨 Пример композиции

**4 слоя на одном экране:**
1. **Фон** (Conway): Ocean палитра, α=100/60, normal
2. **Средний** (HighLife): Fire палитра, α=180/120, screen
3. **Акцент** (Lenia): Neon палитра, α=220/150, additive
4. **Детали** (Seeds): Galaxy палитра, α=160/80, overlay

**Результат**: Все 4 слоя видны одновременно, создавая богатую композицию!

## 🎚️ Управление видимостью

### Solo/Mute система:
- **mute=true**: Слой исключается из рендеринга
- **solo=true**: Показываются только solo слои
- Позволяет изолировать или комбинировать слои

## 🔄 Pipeline рендеринга

1. Очистка canvas
2. Фильтрация слоев (solo/mute)
3. Для каждого слоя:
   - Генерация цветов клеток
   - Применение палитр
   - Применение прозрачности
   - Наложение с учетом blend_mode
4. Постэффекты
5. Отображение на экране

## ✅ Ключевые преимущества

- **ОДИН ЭКРАН**: Все слои проектируются на общий экран
- **КОМПОЗИЦИЯ**: Финальное изображение = сумма всех слоев  
- **ГИБКОСТЬ**: Полный контроль прозрачности и смешивания
- **РЕАЛЬНОЕ ВРЕМЯ**: Все происходит синхронно с аудио
- **ТВОРЧЕСКАЯ СВОБОДА**: Бесконечные возможности комбинирования

Теперь LayerGenerator позволяет создавать сложные многослойные композиции, где каждый слой вносит свой вклад в единое художественное произведение на экране! 🎨✨
