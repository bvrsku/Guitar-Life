# HUD Clear Parameters Documentation

## Добавлены новые параметры очистки в HUD

В интерфейс Guitar Life добавлены два важных параметра управления очисткой популяции клеток:

### 1. Clear Type (Тип очистки)

**Расположение:** В секции "ACTIONS" рядом с кнопкой "Clear"

**Доступные варианты:**
- **Полная очистка** - Удаляет все клетки со всех слоев (по умолчанию)
- **Частичная очистка** - Удаляет заданный процент клеток (настраивается слайдером "Partial Clear %")
- **Очистка по возрасту** - Удаляет клетки старше определенного возраста (настраивается слайдером "Age Threshold")
- **Случайная очистка** - Удаляет случайный процент клеток (настраивается слайдером "Random Clear %")

**Как использовать:**
1. Выберите нужный тип очистки из выпадающего списка "Clear Type"
2. Настройте соответствующие параметры слайдерами:
   - Для частичной: "Partial Clear %" (10-90%)
   - Для возрастной: "Age Threshold" (1-50)
   - Для случайной: "Random Clear %" (10-90%)
3. Нажмите кнопку "Clear" для выполнения очистки

### 2. Soft Mode (Режим мягкой очистки)

**Расположение:** В секции "CONTROLS" рядом с кнопкой "Soft Clear"

**Доступные варианты:**
- **Kill** - Мгновенное удаление клеток (по умолчанию)
- **Fade** - Мягкое затухание яркости клеток
- **Fade+Kill** - Комбинированный режим: затухание + удаление

**Как использовать:**
1. Включите "Soft Clear" кнопкой (должна быть активна)
2. Выберите режим из выпадающего списка "Mode"
3. Настройте параметры контроля популяции:
   - "Max Cells %" - максимальный процент заполнения сетки
   - "Clear At %" - при каком проценте начинать очистку
   - "Age Bias %" - приоритет удаления старых клеток

### Дополнительные настройки

**Слайдеры для точной настройки очистки:**
- **Partial Clear %** (10-90%) - процент клеток для частичной очистки
- **Age Threshold** (1-50) - возрастной порог для очистки по возрасту
- **Random Clear %** (10-90%) - процент клеток для случайной очистки

**Автоматическая система контроля популяции:**
- **Max Cells %** (10-90%) - максимальное заполнение сетки
- **Clear At %** (40-95%) - порог начала автоочистки
- **Age Bias %** (0-100%) - вероятность удаления старых vs случайных клеток

### Статус и мониторинг

В HUD добавлены информационные панели:
- **Population**: Показывает текущее количество клеток и процент заполнения
- **Removal**: Отображает текущий режим удаления и пороги

### Примеры использования

1. **Творческая очистка:** Установите "Частичная очистка" 30% для создания интересных паттернов
2. **Управление возрастом:** Используйте "Очистка по возрасту" с порогом 15 для обновления популяции
3. **Мягкий контроль:** Режим "Fade" для плавного затухания перенаселенных областей
4. **Агрессивный контроль:** Режим "Kill" для быстрого контроля популяции

### Горячие клавиши

- **C** - выполнить очистку выбранного типа
- **R** - создать случайные паттерны
- **T** - создать тестовый паттерн

### Сохранение настроек

Все параметры очистки автоматически сохраняются в конфигурации и восстанавливаются при запуске приложения.
