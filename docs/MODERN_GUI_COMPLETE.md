# Guitar Life v13 - Современный GUI

## Что было создано

Создан полнофункциональный современный GUI для настройки всех параметров Guitar Life v13. 

### Основные возможности GUI:

#### 📱 **Структура интерфейса**
- **Многовкладочный интерфейс** с 6 основными разделами
- **Адаптивный дизайн** с прокручиваемыми элементами
- **Автоматическое сохранение/загрузка** настроек

#### 🎵 **Вкладка "Аудио"**
- Выбор аудио устройства из списка доступных
- Настройка усиления (Gain): 0.1 - 5.0
- Сила RMS: 0 - 200%
- Пороги RMS: минимум и максимум

#### 🎨 **Вкладка "Слои"**
- Количество слоев: 1-5
- Опция "Разные настройки для слоев"
- Поддержка индивидуальных конфигураций

#### ⏱️ **Вкладка "Тайминги"**
- Интервал тика: 10-500 мс
- Адаптивный тик по высоте ноты
- Настройки минимального и максимального тика

#### 🌈 **Вкладка "Цвета"**
- Максимальный возраст клеток: 20-300
- Начало затухания: 10-200
- Автоматические настройки спада насыщенности и яркости

#### ✨ **Вкладка "Эффекты"**
- **Следы (Trails)** с настройкой силы
- **Размытие (Blur)**
- **Свечение (Bloom)**
- **Постеризация**
- **Сканлайны**

#### 🧹 **Вкладка "Очистка"**
- Мягкая очистка с включением/выключением
- Максимальное заполнение: 10-90%
- Порог очистки RMS: 0.001-0.02

### Технические особенности:

#### 🔄 **Интеграция**
- Полная замена старой функции `choose_settings()`
- Автоматическое определение аудио устройств
- Совместимость со всеми существующими параметрами

#### 💾 **Сохранение настроек**
- Автоматическое сохранение в `guitar_config.json`
- Загрузка существующих настроек при запуске
- Поддержка всех параметров приложения

#### 🎛️ **Настройки по умолчанию**
- Предустановленные конфигурации для 3 слоев
- Разные правила автоматов: Conway, HighLife, Day&Night
- Различные палитры для каждого слоя
- Оптимальные настройки для начала работы

### Использование:

1. **Запуск**: `python guitar_life.py`
2. **Настройка**: Откроется GUI с вкладками
3. **Применение**: Нажать "OK" для сохранения настроек
4. **Отмена**: Нажать "Отмена" для использования настроек по умолчанию

### Файлы:

- `modern_gui.py` - Современный GUI интерфейс
- `guitar_config.json` - Файл сохранения настроек (создается автоматически)
- `guitar_life.py` - Основное приложение (обновлено для использования нового GUI)
