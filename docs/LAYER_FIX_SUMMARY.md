## 📊 Техническая реализация:

```python
# Каждый слой обновляется независимо:
for i, L in enumerate(self.layers):
    L.age[L.grid] += 1
    L.age[~L.grid] = 0
    L.grid = step_life(L.grid, L.rule)  # ← Индивидуальное правило

# Каждый слой отрисовывается со своими палитрами:
for L in layers:
    img = build_color_image(L.grid, L.age, L.color_mode, rms, pitch, cfg,
                          L.age_palette, L.rms_palette)  # ← Индивидуальные палитры
    self.renderer.blit_layer(img, L.mix)  # ← Накладывается поверх
```
