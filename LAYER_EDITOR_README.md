## Функциональность:

### **Управление слоями:**
- Динамическое изменение количества слоев (1-4)
- Режим "разные настройки по слоям" vs "одинаковые"
- Автоматическая смена правил и палитр по таймеру
- Индивидуальные настройки каждого слоя

### **Режимы Solo/Mute:**
- **Solo** - показывать только выбранный слой
- **Mute** - скрывать выбранный слой
- Поддержка множественного выбора режимов

### **Палитры и правила:**
- Отдельные палитры для возраста клеток и RMS
- 8 правил клеточных автоматов
- 7 цветовых палитр
- Циклическое переключение через кнопки

### **Зеркалирование:**
- Зеркалирование по осям X и Y
- Применяется ко всем слоям одновременно
